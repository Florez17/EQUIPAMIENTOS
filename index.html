Acerca de este mapa

Este mapa interactivo muestra los lugares de interés en La Ceja, Antioquia, Colombia.

¿Cómo usar el mapa?

  • Usa el menú desplegable para seleccionar un lugar específico
  • Filtra por categorías para ver lugares similares
  • Usa la búsqueda libre para encontrar lugares por nombre o descripción
  • Haz clic en "Mostrar todos" para ver todos los lugares en el mapa
  • Usa "Mi ubicación" para centrar el mapa en tu posición actual
